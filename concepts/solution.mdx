zkVerify solves confidential browsing by combining a private data channel (zkTLS), local proving (zkVMs), and scalable verification (aggregation + zkVerify settlement).

## Browser extension as a ZK sidecar

The extension:

- Uses **zkTLS** (e.g. TLSNotary) to fetch authentic web data privately
- Uses **lightweight zkVMs** (RISC Zero or SP1 compiled to WASM) to generate proofs locally
- Submits proofs for **aggregation** and ultra-fast verification on **zkVerify** (**< 1 second**)
- Receives an **on-chain attestation** that a website can query via simple APIs

## What websites get

- A simple **attestation check** (instead of integrating each proof system)
- **Chain-agnostic settlement** (works across L1/L2)
- Minimal user friction: “prove once, reuse”
