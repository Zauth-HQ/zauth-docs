<Info>
  **Prerequisites**: Node.js 19+ and a local clone of this docs repo with a `docs.json` file.
</Info>

## Preview docs locally

<Steps>
  <Step title="Install Mintlify CLI">

```bash
npm i -g mint
```

  </Step>
  <Step title="Run the dev server">

From the directory that contains `docs.json`:

```bash
mint dev
```

Open **http://localhost:3000** to view the docs. Changes to MDX files update automatically.

  </Step>
</Steps>

## Custom port

```bash
mint dev --port 3333
```

## Validate links

```bash
mint broken-links
```

## Working on the validator project (task/)

The documentation in this repo describes the **ERC-8004 Mandate Validator** app. To run or develop the app itself:

1. Go to the `task/` directory (sibling or wherever the validator code lives).
2. Follow the [Quickstart](/quickstart) for install, `.env`, and `npm run start` / `npm run trust` / `npm run server`.

For Mintlify docs customization (theme, nav, etc.), see [Mintlify documentation](https://mintlify.com/docs).
